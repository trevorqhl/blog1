Login as root
{% highlight ruby %}
#=> Install RPMs
yum install -y ruby ruby-devel make gcc gcc-c++ redhat-rpm-config
gem install jekyll bundler
#=> Install Jekyll
jekyll new blog
bundle exec jekyll serve
{% endhighlight %}
Push the blog to Github
